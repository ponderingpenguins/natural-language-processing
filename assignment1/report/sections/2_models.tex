We implemented two models for the text classification task: Logistic Regression and Linear Support Vector Machine (SVM), both using Term Frequency-Inverse Document Frequency (TF-IDF) for feature extraction. The architecture of both models is similar, with the main difference being the linear classifier used for classification. The input text is first preprocessed and then transformed into TF-IDF features, which are then fed into either a Logistic Regression classifier or a Linear SVM for classification (see Figure \ref{fig:model_architecture} for a visual representation of the architecture).
