Dataset for this text classification by topic task is the textual AG News dataset from Hugging Face. Each data point has an associated label, the new's title and its description. There are 4 classes, all of which has 30000 training and 1900 testing samples: world, sports, business, and sci/tech.

% Split details
% split sizes
% split seed

The dataset is split into 3: Training, test, and dev. Using the official split from Hugging Face, training and test sets were pre-split. We generated the dev set from the training set (90/10).
The split seed is 67.
