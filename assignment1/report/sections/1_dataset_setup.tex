Dataset for this text classification by topic task is the textual AG News dataset from Hugging Face \cite{zhang2015character}. Each data point has an associated label, the news title and its description. There are 4 classes, all of which has 30000 training and 1900 testing samples: world, sports, business, and sci/tech.

% Split details
% split sizes
% split seed

We use the official Hugging Face split (120,000 train and 7,600 test).
We use stratified sampling (90/10) with seed 67 to create a development set:
\begin{itemize}
    \item Train: 108,000
    \item Dev: 12,000
    \item Test: 7,600
\end{itemize}

\subsection{Preprocessing}
% Preprocessing (Normalization & Tokenization)
% Lowercasing
% Removing whitespace
% Removing punctuation
The input is the title and description concatenated. We used the following preprocessing steps:
\begin{itemize}
    \item Lowercasing
    \item Whitespace removal
    \item Punctuation removal
    \item English stopword removal (using NLTK library)
    \item Lemmatization (with WordNetLemmatizer)
\end{itemize}

The reason why we remove stop words, is because common words often carry less information and carry noise into our classifier.

% Lemazation (vs. stemming)
The last step in the normalization is lemmatization, the act of reducing words into their dictionary form (e.g. "running" to "run"). Using the NLTK's WordNetLemmatizer, we reduced each word in the text to their dictionary forms to reduce the computational complexity. In this way, different forms of the same word are reduced to a single form.

% Vectorization through IT-IDF

\subsection{Tokenization}
Even though there is no explicit tokenization in our code, texts are vectorized with word-level TF-IDF. After the normalization steps, TF-IDF vectorizes, and thus tokenizes, the input text into numerical data:
\begin{itemize}
    \item n-gram range: (1, 2)
    \item \verb|max_features| = 10,000
    \item \verb|min_df| = 2
\end{itemize}

\subsection{Reproducibility}
All experiments make use of the fixed random seed 67.
We use it for the data split and model initialization where it is applicable.
