
# Assignment 1

## Setup

```bash
cd assignment1/src
```

```bash
source .venv/bin/activate
```

```bash
uv sync
```

## To run

Make sure to be in the assignment1 directory

```bash
cd assignment1
```

Activate the environment

```bash
source src/.venv/bin/activate
```

Run the main file as a module (allows relative imports)

```bash
python -m src.main
```
