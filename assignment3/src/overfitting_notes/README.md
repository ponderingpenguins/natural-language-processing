# LSTM Overfitting Notes

This folder stores notes generated by `run_lstm_overfitting_experiment.py`.

Each file records:

- run name (`experiment.name`)
- key config values
- train/dev/test sizes
- loss gap summary
- dev/test metrics

Main files used in this assignment:

- `updated_default_full.md` (final full-data LSTM run)
- `summary.md` (consolidated ablation notes)
